== FEATURES

A JS version of the classic snake game on Nokia's. Built in my spare time during Week 5 of NEXT Academy's webdev course.

<b>Features include:</b>

* Snake movement using arrow keys.

* Eat food to grow longer

* Superfood every 10 foods, with bonus points based on timer.

* Crashing into self or walls equal death.

* Restart and pause game.
